# linux-cmd-cheatsheet

## Git command
* Show the working tree status
```
git status
```
*  Show changes between commits, commit and working tree
```
git diff
```
* Change content locally and push
```
git add .
git commit -m "the comments"
git push
```

## Shell script

* change directory
```
cd
```
* show all files in a directory
```
ls
```
